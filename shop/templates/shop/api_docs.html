TerraScope Geospatial Intelligence API

RESTful API for managing real-time satellite observation data with advanced geospatial intelligence capabilities

RESTful Design JWT Authentication JSON Format Real-time Data
API Base URL
{{ api_base_url }}

Authentication

All API requests require JWT Bearer token authentication. Include your token in the Authorization header of every request.

How to Get Your Token
  1. Purchase a TerraScope product from our Products page
  2. Generate an API token from your dashboard
  3. Include the token in the Authorization header of your requests
Authentication Example
bash 
curl -H "Authorization: Bearer YOUR_TOKEN_HERE" \
     {{ api_base_url }}/api/observations

API Endpoints

GET /health

Check API status (no authentication required)

Response 
{"status": "ok"}
GET /api/observations

Retrieve observations with optional filtering

Query Parameters
start_timestamp ISO 8601 datetime - Filter by start date
end_timestamp ISO 8601 datetime - Filter by end date
min_lat / max_lat Number - Latitude range filter
min_lon / max_lon Number - Longitude range filter
Example Request
bash 
curl -H "Authorization: Bearer TOKEN" \
     "{{ api_base_url }}/api/observations?min_lat=50&max_lat=52"
POST /api/observations

Create a new observation record

Request Body
json 
{
  "timestamp": "2025-12-06T12:00:00Z",
  "timezone": "UTC",
  "latitude": 51.5074,
  "longitude": -0.1278,
  "satellite_id": "SAT-001",
  "spectral_indices": {
    "ndvi": 0.75,
    "ndwi": 0.45
  },
  "notes": "Clear sky observation"
}
GET /api/observations/{id}

Retrieve a specific observation by ID

PUT /api/observations/{id}

Replace entire observation

PATCH /api/observations/{id}

Partially update observation

Note: Cannot modify records before the current quarter
DELETE /api/observations/{id}

Delete an observation (current quarter only)

POST /api/observations/bulk

Create multiple observations in a single request

Code Examples

Python (requests)
python 
import requests

TOKEN = "your_jwt_token_here"
BASE_URL = "{{ api_base_url }}"

headers = {
    "Authorization": f"Bearer {TOKEN}",
    "Content-Type": "application/json"
}

# List observations
response = requests.get(f"{BASE_URL}/api/observations", headers=headers)
print(response.json())

# Create observation
data = {
    "timestamp": "2025-12-06T12:00:00Z",
    "timezone": "UTC",
    "latitude": 51.5074,
    "longitude": -0.1278,
    "satellite_id": "SAT-001"
}
response = requests.post(
    f"{BASE_URL}/api/observations",
    headers=headers,
    json=data
)
print(response.json())
JavaScript (fetch)
javascript 
const TOKEN = 'your_jwt_token_here';
const BASE_URL = '{{ api_base_url }}';

// List observations
fetch(`${BASE_URL}/api/observations`, {
    headers: {
        'Authorization': `Bearer ${TOKEN}`
    }
})
.then(res => res.json())
.then(data => console.log(data));

// Create observation
fetch(`${BASE_URL}/api/observations`, {
    method: 'POST',
    headers: {
        'Authorization': `Bearer ${TOKEN}`,
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        timestamp: '2025-12-06T12:00:00Z',
        timezone: 'UTC',
        latitude: 51.5074,
        longitude: -0.1278,
        satellite_id: 'SAT-001'
    })
})
.then(res => res.json())
.then(data => console.log(data));
